Form Authoring with XLSForms

KEDCO Surveys uses XLSForm standard to allow you to author forms in XLS.

XLSForm is a form standard created to help simplify the authoring of forms using Excel. Authoring is done in a human readable format using a familiar tool that almost everyone knows - Excel. XLSForms provide a practical standard for sharing and collaborating on authoring forms. They are simple to get started with but allow for the authoring of complex forms by someone familiar with the syntax.

For a full reference please use: XLSForm.org. Also, be sure to checkout the form gallery which provides examples you can download and modify.
